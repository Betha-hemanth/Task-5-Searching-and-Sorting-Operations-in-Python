5.4 Patient ID Sorting and Search

Scenario: Patient IDs stored in a list need to be sorted and searched.

Algorithm:

1. Store patient IDs in a list.

2. Sort the list.

3. Input patient ID to search.

4. Use binary search to locate the patient ID.

output:

sorted patient list: [101, 102, 105, 107, 108, 110, 115, 120]

enter patient_id: 110

patient_id found at index 5
